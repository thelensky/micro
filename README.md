# Микросервисная платформа (учебный проект)

Проект создан в рамках изучения книги  
**«Микросервисы Spring в действии»** (Джон Карнелл, Иллари Уайлупо Санчес).

**Цель:** освоить на практике модель микросервисной архитектуры с использованием экосистемы Spring.

**Технологии:**
- Java 17
- Spring Boot 4.0.5
- Spring Cloud 2025.0.x (Gateway, Config, Netflix/Eureka, LoadBalancer)
- Resilience4j
- Docker / Docker Compose
- (планируется: Kafka, Kubernetes, Keycloak, Prometheus, Grafana)

**Сборка:** Gradle

**Статус:** в процессе изучения (реализованы главы 1–8, далее по плану)
